# Gym Management System

A Gym Management System developed using Python and MySQL.

## Requirements

* Python 3.x
* MySQL Server
* mysql-connector-python

## Setup

1. Install the required package:

pip install -r requirements.txt

2. Open MySQL and run:

schema.sql

3. Update the database credentials in:

gym_management.py

4. Run the program:

python gym_management.py

## Features

* Add members
* Delete members
* Modify membership plans
* Display all members
* Filter members by gender
