# Cafe Management System

A Python-based cafe management system developed as a Class 11 school project.

## Features

* Displays menu items by category
* Accepts customer orders
* Calculates order totals automatically
* Supports multiple customers
* Maintains order history
* Validates customer contact numbers

## Technologies Used

* Python

## Author

* Arya Rana

## How to Run

1. Install Python.
2. Download the project.
3. Run:

python cafe_management.py

4. Follow the on-screen instructions.
